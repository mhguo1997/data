This repository contains data sets of structures mentioned in Journal of Physical Chemistry A paper “Ab-initio Molecular Dynamics Study on the Interlayer Bonding of Few-Layer Graphene under Pressures” at https://pubs.acs.org/doi/10.1021/acs.jpca.********

The 7z package can be unpacked by 7zip or WinRAR software.

Data.7z contains source data for all structures mentioned in the paper.
